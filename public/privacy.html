Privacy Policy

Last updated: March 2026

Celebrity Game is a browser-based party game. This policy explains what (very little) data is involved when you play.

What we collect

The only information you provide is a player name when joining a room. That name is held in server memory solely to run the game session. It is never written to a database or disk.

How long data is kept

Room data — including all player names and celebrity names — is deleted from memory as soon as all players have left the room. Nothing persists beyond the active session.

What we do not do

  • No user accounts or registration.
  • No cookies or tracking pixels.
  • No third-party analytics (no Google Analytics, Mixpanel, etc.).
  • No advertising.
  • No sale or sharing of any data with third parties.

Hosting

The game server is hosted on Render.com. Render may retain standard infrastructure logs (IP addresses, request timestamps) as part of normal server operation. These logs are subject to Render's own privacy policy.

Children

This game does not knowingly collect any information from children under 13.
